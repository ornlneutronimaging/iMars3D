step 3: supporess noise¶

Noise reduction can help remove streaky type artifacts in the final reconstruction, but the bilateral method (default) might take some time to finish. If the overall runtime for integration test is too long, we can skip this step as we don't really care about the niceness/cleaness of the final reconstruction results during integration test.
