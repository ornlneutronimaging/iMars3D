Welcome to iVenus!
==================

`iVenus <https://github.com/ornlneutronimaging/iVenus.git>`_ is an 
open-sourced Python toolbox to Neutron data processing and image processing
tasks at the `Oak Ridge National Laboratory <http://www.ornl.gov>`_.

This guide is maintained on 
`GitHub <https://github.com/ornlneutronimaging/iVenus/tree/master/doc>`_.
